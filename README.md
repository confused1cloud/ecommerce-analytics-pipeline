\# 🛒 E-Commerce Analytics Pipeline



\## 📊 Overview

A complete end-to-end data analytics pipeline for e-commerce data.



\*\*Key Features:\*\*

\- Generates realistic e-commerce data (customers, products, orders)

\- ETL pipeline with RFM customer segmentation

\- Interactive Streamlit dashboard with 5+ visualizations



\## 🛠️ Tech Stack

\- \*\*Python\*\* - Data processing

\- \*\*Pandas\*\* - Data manipulation

\- \*\*PostgreSQL\*\* - Data warehouse

\- \*\*SQLAlchemy\*\* - Database connection

\- \*\*Streamlit\*\* - Interactive dashboard

\- \*\*Plotly\*\* - Data visualization
